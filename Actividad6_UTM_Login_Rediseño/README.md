# Actividad 6 - Rediseño Responsivo

Rediseño responsivo de la página de inicio (login) de la plataforma institucional de la Universidad Técnica de Manabí.

## Tecnologías utilizadas
- HTML5
- CSS3
- Flexbox
- Media Queries

## Descripción
Este proyecto mejora la adaptabilidad, legibilidad y experiencia de usuario en dispositivos móviles, tablet y escritorio.
